# NEXVAULT - secure software 

## the purpose: 

Create a functional and interactive website for an antivirus sales company 

## Technology stack 

- **Languages:** Html, CSS, JavaScript

## System requirements
- Linux|MacOS|Windows

## Quick setup

Follow these steps to run the project locally:

1. **Clone the repository**
2. **Install Node.js and npm**
3. **Install Live Server (optional but recommended)**
  **You can install the Live Server extension if you use VS Code:**
      Open VS Code → Extensions → search "Live Server" → Install

4. **Open the project html folder**
5. **Right-click nexVaultIndex.html → "Open with Live Server"**
